== Installation ==

1. Copy the 'jigoshop-woocommerce-store-migrator' zip on your desktop. 
2. Unzip it.
3. Upload the folder 'jigoshop-woocommerce-store-migrator' to the '/wp-content/plugins/' directory.
4. Activate 'jigoshop-woocommerce-store-migrator' through the 'Plugins' menu in WordPress.

That's it!

== Usage ==

1. De-activate Jigoshop if you haven't already.
2. Install and/or activate WooCommerce.
3. Open 'WooCommerce > Store Migrator'.
4. Review migration details.
5. Jump to the 'Migrate' tab or click on the 'Jump To Migrate' button. 
6. Then click on the 'Migrate' button.

Done!
